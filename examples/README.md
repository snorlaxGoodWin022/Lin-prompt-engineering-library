# 使用示例

通过实际示例展示 Prompt 模板的使用效果。

## 示例分类

### 生成的组件
- **generated-components/** - AI 生成的组件代码示例
  - React 组件示例
  - Vue 组件示例
  - 组件文档示例

### 优化报告
- **optimization-reports/** - 性能优化报告示例
  - Lighthouse 优化方案示例
  - Bundle 分析优化示例
  - 图片优化方案示例

## 使用目的

1. **模板验证**：展示模板的实际生成效果
2. **质量参考**：提供高质量的生成结果作为参考
3. **学习示例**：帮助用户理解如何有效使用模板
4. **效果评估**：评估不同模板的生成质量差异

## 如何添加新示例

1. 使用模板生成代码或文档
2. 将生成结果保存到对应分类目录
3. 添加说明文件解释：
   - 使用的模板名称
   - 输入的参数
   - 生成效果评估
   - 可能的改进点

## 示例标准

- **真实性**：基于实际使用场景生成
- **完整性**：包含完整的输入和输出
- **可读性**：良好的格式和注释
- **实用性**：可以直接参考使用的示例

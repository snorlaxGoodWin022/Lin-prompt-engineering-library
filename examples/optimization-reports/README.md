# 优化报告示例

通过性能优化模板生成的优化方案示例。

## 示例类型

### Lighthouse 优化报告
- 基于 Lighthouse 得分的具体优化方案
- 包含问题分析、解决方案和预期效果

### Bundle 分析报告
- Bundle 分析结果和代码分割方案
- 第三方库优化和公共 chunks 提取方案

### 图片优化方案
- 图片资源分析结果
- 压缩、格式转换、懒加载具体方案

## 示例结构

每个示例应包含：
1. **原始数据**：Lighthouse 报告、Bundle 分析图等原始数据
2. **输入参数**：填充到模板中的项目信息
3. **生成方案**：AI 生成的完整优化方案
4. **实施效果**：实际实施后的效果验证（如可能）
5. **方案评估**：方案的可行性、成本和效果评估

## 使用价值

1. **方案参考**：提供可直接参考的优化方案
2. **效果预估**：帮助预估优化工作的投入产出比
3. **技术学习**：学习系统化的性能优化方法
4. **模板验证**：验证模板生成方案的质量和实用性

# AI 生成的组件示例

通过 Prompt 模板生成的 React/Vue 组件代码示例。

## 示例结构

每个示例应包含：
1. **使用的模板**：模板文件名和版本
2. **输入参数**：填充到模板中的具体参数
3. **生成代码**：AI 生成的完整代码
4. **效果评估**：生成代码的质量、完整性和可用性评估
5. **改进建议**：如果需要手动调整的地方

## 示例目的

1. **展示效果**：直观展示模板的生成能力
2. **建立信心**：证明模板能生成高质量代码
3. **提供参考**：作为用户自己使用模板的参考
4. **持续改进**：收集反馈优化模板

## 贡献指南

欢迎提交新的生成示例：
1. 使用现有模板生成组件代码
2. 评估生成代码的质量
3. 创建示例文件夹，包含：
   - `input-params.md`：输入的参数详情
   - `generated-code/`：生成的代码文件
   - `evaluation.md`：生成效果评估
4. 提交 Pull Request

# 性能优化模板

针对前端性能优化场景的 Prompt 模板。

## 可用模板

### Lighthouse 分析
- **lighthouse-analysis.md** - Lighthouse 报告解读与优化方案生成

### Bundle 分析
- **bundle-analysis.md** - Bundle 分析、代码分割和体积优化方案

### 图片优化
- **image-optimization.md** - 图片资源压缩、格式转换和懒加载方案

## 使用场景

1. **性能审计**：网站性能评分低，需要系统优化
2. **加载优化**：首屏加载时间过长，需要代码分割
3. **资源优化**：图片、字体等资源体积过大
4. **构建优化**：打包产物体积需要缩减

## 模板特点

- **数据驱动**：基于 Lighthouse 得分、Bundle 分析数据等
- **方案具体**：提供可直接实施的代码和配置
- **优先级排序**：区分快速优化和长期重构任务
- **效果预估**：给出优化后预期提升效果

## 最佳实践

1. 使用前收集准确的性能数据（Lighthouse 报告、Bundle 分析图）
2. 根据项目特点调整优化方案（如是否需要 SEO、用户网络环境）
3. 优先实施高优先级、低成本的优化
4. 优化后重新测试验证效果

# Prompt 模板库

所有 Prompt 模板按场景分类存放于此目录。

## 分类说明

### components/ - 组件开发模板
用于生成 React、Vue 等前端组件的 Prompt 模板，包含完整的技术栈规范、代码结构要求和组件需求定义。

### performance/ - 性能优化模板
针对 Lighthouse 分析、Bundle 分析、图片优化等性能优化场景的 Prompt 模板。

### collaboration/ - 团队协作模板
用于生成 Git Commit Message、Pull Request 描述、技术文档等团队协作场景的 Prompt 模板。

### general/ - 通用模板
代码审查、调试帮助等通用开发场景的 Prompt 模板。

## 使用说明

1. 选择适合的场景目录
2. 查看对应的模板文件
3. 复制模板内容，替换 `{...}` 占位符
4. 与 AI 助手使用生成代码或文档

## 模板结构

每个模板通常包含以下部分：
- **技术栈/Technology Stack**：指定使用的框架、库和工具
- **代码规范/Code Standards**：命名规范、文件结构、编码规则
- **需求定义/Requirements**：具体的功能需求、Props/Events 定义
- **额外要求/Additional Requirements**：错误处理、测试、性能等要求

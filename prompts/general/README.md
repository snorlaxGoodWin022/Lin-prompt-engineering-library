# 通用模板

适用于多种开发场景的通用 Prompt 模板。

## 可用模板

### 代码审查
- **code-review.md** - 代码审查提示，帮助发现潜在问题

### 调试帮助
- **debugging-help.md** - 调试问题分析和解决方案建议

## 使用场景

1. **代码审查**：系统化检查代码质量，发现潜在问题
2. **问题调试**：分析错误日志，定位问题原因
3. **代码优化**：改进现有代码结构和性能
4. **知识查询**：快速获取技术概念和最佳实践

## 模板特点

- **场景灵活**：适用于多种开发任务
- **结构清晰**：提供系统化的分析框架
- **实用导向**：注重实际问题的解决方案
- **学习辅助**：帮助开发者提升技能

## 最佳实践

1. 根据具体问题调整模板中的问题描述
2. 提供足够的上下文信息（错误日志、代码片段等）
3. 明确期望的输出格式（代码、文档、分析报告等）
4. 将成功解决问题的 Prompt 保存为新的模板

# 组件开发模板

用于生成 React、Vue 等前端组件的 Prompt 模板。

## 可用模板

### React 组件模板
- **react-component.md** - 标准 React + TypeScript 业务组件生成模板
- **component-documentation.md** - 组件文档自动生成模板

### Vue 组件模板
- **vue-component.md** - Vue 3 组合式 API 组件生成模板

## 使用场景

1. **新组件开发**：快速生成符合项目规范的组件代码
2. **组件重构**：将现有组件升级到新的技术栈或规范
3. **文档生成**：为已有组件自动生成使用文档
4. **团队培训**：新成员学习项目组件规范

## 模板特点

- **技术栈明确**：指定 React/Vue 版本、UI 库、状态管理方案
- **结构规范**：定义文件夹结构、文件命名规则
- **类型安全**：要求完整的 TypeScript 类型定义
- **质量要求**：包含错误处理、测试、代码注释等要求

## 最佳实践

1. 使用前确认模板中的技术栈与项目一致
2. 根据实际需求调整 Props/Events 定义
3. 生成的代码仍需人工 Review 确保质量
4. 定期更新模板以匹配项目技术栈升级

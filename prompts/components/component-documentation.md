## 文档要求

1. **组件简介** - 一句话说明用途
2. **安装依赖说明** - 需要安装的包和版本
3. **Props API 表格**
   ```
   | 参数名 | 类型 | 默认值 | 说明 |
   |--------|------|--------|------|
   | ...    | ...  | ...    | ...  |
   ```
4. **Events 事件表格**（如果有）
   ```
   | 事件名 | 参数 | 说明 |
   |--------|------|------|
   | ...    | ...  | ...  |
   ```
5. **三个使用示例**
   - **基础用法** - 最简单的使用方式
   - **进阶用法** - 复杂场景的使用方式
   - **完整配置示例** - 展示所有可用配置
6. **注意事项** - 如果有特殊限制或使用建议
7. **FAQ** - 至少 3 个常见问题及解答

## 输出格式

- 使用 Markdown 格式
- 结构清晰，便于阅读
- 代码示例使用代码块
- 表格对齐整齐
- 标题层级合理
